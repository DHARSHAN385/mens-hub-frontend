
  # mens hub front end

  ## Project Overview
  Full-stack ecommerce app for men's fashion. Includes React frontend (Vite), Django backend (REST API), MySQL DB, and full documentation.

  ### Figma Design
  [Figma Project Link](https://www.figma.com/design/o5jBxYvsL0YNUhiBQdio7X/mens-hub-front-end)

  ## 🚀 Quick Start

  ### Frontend
  1. Install dependencies: `npm i` or `pnpm install`
  2. Start dev server: `npm run dev` or `pnpm dev`

  ### Backend
  1. Install Python deps: `pip install -r requirements.txt`
  2. Set up MySQL (see BACKEND_README.md)
  3. Run migrations: `python manage.py migrate`
  4. Start server: `python manage.py runserver`
